# About The Project ‚úå
The application is a scientific programmable calculator that can handle complex number and a series of operations on them. The application shows a modern GUI to insert and compute operation on Numbers.

 </p>

# Key Features ‚ú®
 ‚Ä¢ It is possible to enter complex numbers in the format a + bj using the keyboard or the GUI.
 
 ‚Ä¢ The operations supported on complex numbers are +, -, *, /, sqrt to calculate the root, +- which reverses the sign, mod to calculate the modulus, arg to calculate the argument, exp to calculate the exponential, log to calculate the natural logarithm, sin for the sine, cos for the cos, tan for the tangent.

 ‚Ä¢ It is possible to modify the stack using the commands drop, which removes the element in the dup stack, which makes a copy of the last element stack, swap, which \ n swaps the last two elements of the stack; above what a copy of the penultimate element of the stack; or you can type the clear command to delete all elements from the stack.
   
 ‚Ä¢ By clicking the inv key, it is also possible to take advantage of the inverse trigonometric functions

 ‚Ä¢ It is possible to use 26 Variables form "a" to "z" 
 
 ‚Ä¢ '>x' to insert the value at the top of the stack into the variable.
 
 ‚Ä¢ '<\x' to insert the value into the variable at the top of the stack.
 
 ‚Ä¢ '+x' to add the value at the top of the stack to the variable.
 
 ‚Ä¢ '-x' to subtract the value at the top of the stack from the variable.
 
 ‚Ä¢ By pressing the Expand key you can enter user defined operations.
 
 ‚Ä¢ User defined operations are defined by a name and a list of instuction separated by a comma. The user can take advantage of all the operations, can put a number in the stack, or save and restore the variables.
 
 ‚Ä¢ The User defined operations can be removed by typing the name of the operation to be deleted and clicking on the remove button or by selecting an operation from the table and right clicking.
 
 ‚Ä¢ The programmable operations can be modified by typing the name of the operation to be deleted and clicking on the modify button or by double clicking on an operation directly from the table.

 ‚Ä¢ The user can save the User Defined Operations by clicking on the export button in file bar and choose to export them in CSV format or Dat Format

 ‚Ä¢ The user can save the variables stored entering the "save" command or using the menu in the file bar.

 ‚Ä¢ The user can restore the variables in the variable stack entering the "restore" command or using the menu in the file bar.

# Demo üìà
If you want to start to use the calculator, you must insert inside the text area a complex number in cartesian notation, or using the keyboard and than press enter, or using the bottons present on the interface and than press "=". Than the number is shown in the stack table, in this way it is possible to continue performing the operations on the numbers present in the stack (For the operation that require two operands it's necessary that the stack has at least two elements).
